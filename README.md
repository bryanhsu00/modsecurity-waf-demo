# modsecurity-waf-demo

## Introduction
An implementation of a WAF proxy server sitting front of a vulnerable website(DVWA)

## Usage
```
docker-compose up
```

## Components
- **etc**: modsecurity and nginx config files
- **modsec_audit.log**: modsecurity log file
